## Overview
Interactive Power BI dashboard analyzing Netflix content distribution, ratings, genres, release trends, and geographic availability using publicly available datasets.

## Tools Used
- Power BI
- Power Query
- Excel

## Key Features
- Content growth trend analysis
- Genre and rating distribution
- Geographic content mapping
- Movie and TV show breakdown
- Release year analysis
- Interactive filtering and visualization

## Business & Analytical Insights
- Identified top-performing content categories and genre trends
- Analysed regional content distribution and production patterns
- Visualized Netflix content expansion over time
- Developed executive-style dashboard layouts focused on readability and reporting clarity
