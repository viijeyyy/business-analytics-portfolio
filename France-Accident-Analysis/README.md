## Overview
Interactive Power BI dashboard analyzing French road accident data from 2005–2016 to identify demographic, geographic, seasonal, and vehicle-related trends.

## Tools Used
- Power BI
- Power Query
- Excel

## Key Features
- Accident severity analysis
- Vehicle involvement insights
- Weather condition analysis
- Time-period and seasonal trend tracking
- Geographic and demographic reporting
- Interactive slicers and drill-through navigation

## Business & Analytical Insights
- Identified high-risk time periods and seasonal accident patterns
- Analysed demographic and vehicle-related accident distributions
- Visualized weather-related accident trends for insight generation
- Developed multi-page reporting structure for easier data interpretation
